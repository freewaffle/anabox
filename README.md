# Anabox

Anabox is a top-down 2D shooter game with focus on chaos and destruction.

# Structure

The game is split into these directories:

- `components` folder contains important game's components.
- `dmdata` folder contains files and manifests neccesary for proper building and game work.
- `gamesys` folder contains the engine that powers all Anabox levels, built with Defold components.
- `gdata` folder contains files that being read by the game in runtime. They are not related to Defold engine.
- `maps` folder contains... maps?
- and `resources` folder contains assets, most of them built from project files from `assets` folder.
